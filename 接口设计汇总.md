# 接口设计文档
# 公共部分通用说明
## 1. 账户与认证模块



### 1.2 登录系统

**接口功能**：用户登录系统，根据角色返回不同的登录结果

**请求URL**：`/api/login`

**请求方法**：`POST`


**角色要求**: 无（公开）
**请求参数**：

```json
{
  "userName": "string",
  "password": "string"
}
```

```
| 参数名   | 类型   | 是否必填 | 说明                 |
| -------- | ------ | -------- | -------------------- |
| userName | String | 是       | 邮箱（作为登陆账号） |
| password | String | 是       | 密码                 |

**响应格式**：

```json
{
  "code": 200,
  "message": "登录成功",
  "data": {
    "token": "string",
    "userId": 1,
    "userName": "string",
    "role": "user"
  }
}
```





**错误响应示例**：

```json
{
  "code": 401,
  "message": "用户名或密码错误",
  "data": null
}
```

**主要错误码**：

- 200: 登录成功
- 401: 用户名或密码错误
- 400: 参数错误



---

### 10.1 统一响应格式

所有接口统一使用以下响应格式：

```json
{
  "code": 200,
  "message": "操作成功",
  "data": {}
}
```

### 10.2 统一错误码

- 200: 操作成功
- 400: 请求参数错误或业务逻辑错误
- 401: 未登录或token过期
- 403: 无权限（如非管理员访问管理员接口）
- 404: 资源不存在
- 500: 服务器内部错误

### 10.3 认证方式

所有需要登录的接口（除登录接口外）需要在请求头中携带JWT token：

```
Authorization: Bearer {token}
```

### 10.4 时间格式

- 日期格式：`YYYY-MM-DD`
- 时间格式：`HH:mm:ss`
- 日期时间格式：`YYYY-MM-DD HH:mm:ss`

### 10.5 分页说明

分页查询接口统一使用以下参数：

- `page`: 页码，从1开始
- `size`: 每页数量

分页响应格式：

```json
{
  "total": 100,
  "page": 1,
  "size": 10,
  "list": []
}
```

## 2. 图书检索模块

### 2.1 查看所有图书

**接口功能**：分页查看馆藏图书列表，支持基础浏览

**请求URL**：`/api/books`

**请求方法**：`GET`

**请求参数**：

| 参数名 | 类型 | 是否必填 | 说明 |
|--------|------|----------|------|
| page | Number | 否 | 页码，从 1 开始，默认 1 |
| size | Number | 否 | 每页数量，默认 10 |

**响应格式**：
```json
{
  "code": 200,
  "message": "操作成功",
  "data": {
    "total": 100,
    "page": 1,
    "size": 10,
    "list": [
      {
        "id": "string",
        "title": "string",
        "author": "string",
        "isbn": "string",
        "genre": "string",
        "cover": "string",
        "available": true,
        "availableCopies": 3,
        "createdAt": "YYYY-MM-DD HH:mm:ss"
      }
    ]
  }
}
```

**主要错误码**：
- 200: 成功
- 400: 参数错误

---

### 2.2 根据条件查找图书（按书名 / 作者搜索）

**接口功能**：按书名或作者搜索图书，快速找到感兴趣的书或某作者作品

**请求URL**：`/api/books/search`

**请求方法**：`GET`

**请求参数**：

| 参数名 | 类型 | 是否必填 | 说明 |
|--------|------|----------|------|
| keyword | String | 是 | 搜索关键词（匹配书名或作者） |
| type | String | 否 | 搜索类型：`title`（书名）/ `author`（作者），不传则同时匹配书名与作者 |
| page | Number | 否 | 页码，默认 1 |
| size | Number | 否 | 每页数量，默认 10 |

**响应格式**：与 2.1 一致，`data` 为分页结构，`list` 为图书简要信息数组。

**主要错误码**：
- 200: 成功
- 400: 缺少 keyword 或参数错误

---

### 2.3 搜索筛选（按图书分类 / 语言）与排序（按日期 / 评分）

**接口功能**：在检索结果中按分类、语言筛选，并按上架日期或评分排序，更快找到合适的书

**请求URL**：`/api/books`

**请求方法**：`GET`

**请求参数**：

| 参数名 | 类型 | 是否必填 | 说明 |
|--------|------|----------|------|
| keyword | String | 否 | 搜索关键词（可选，与 2.2 组合使用） |
| genre | String | 否 | 图书分类筛选：`Technology` / `Fiction` / `Science` / `History` / `Management` |
| language | String | 否 | 语言筛选：`Chinese` / `English` / `Others` |
| sortBy | String | 否 | 排序字段：`createdAt`（上架日期）/ `rating`（评分） |
| sortOrder | String | 否 | 排序方向：`asc` / `desc`，默认 `desc` |
| page | Number | 否 | 页码，默认 1 |
| size | Number | 否 | 每页数量，默认 10 |

**响应格式**：与 2.1 一致；若按评分排序，`list` 中每项可包含 `averageRating` 字段。

**主要错误码**：
- 200: 成功
- 400: 参数错误

### BookCopy 创建逻辑

添加图书时，系统会自动根据 `availableCopies` 数量创建对应数量的图书副本：

| 字段 | 说明 |
|------|------|
| barcode | 格式：`{isbn}-{001, 002, ...}`，如 `7504322865-001` |
| bookId | 关联的图书 ID |
| shelfLocation | 书架位置（继承自图书） |
| available | 默认 true（可借） |

**示例**：
- ISBN: `7504322865`，availableCopies: `3`
- 创建 3 条 BookCopy 记录：
  - `barcode: "7504322865-001"`
  - `barcode: "7504322865-002"`
  - `barcode: "7504322865-003"`

**响应格式** - 失败：
```json
{
  "code": 400,
  "message": "Invalid ISBN or barcode format",
  "data": null
}
```

```json
{
  "code": 404,
  "message": "Book not found with this ISBN",
  "data": null
}
```

**主要错误码**：
- 200: 成功找到图书
- 400: 无效的 ISBN/条形码格式
- 404: 图书不存在
- 401: 未登录或无权限


### 1.2 验证规则

**ISBN 格式验证**：
- ISBN-10：10位数字，最后一位可以是数字或X
- ISBN-13：13位数字
- 条形码：最多20位字母数字字符
